# Library #


Web application based on Python, Flask and React for create library by Lib.rus.ec zip-archives.


# Requirements #

* MongoDB

* Redis

* Python 3.5-3.7 supported.

* Flask 1.11-2.2 suppported.

* NodeJs 10 supported.

* Yarn 1.15

# Install #

1. create virtualenv 

2. pip install -r requirements.txt

3. set settings

4. run tools/loader

5. run run.py
